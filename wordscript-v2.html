Common Mistakes

Variable not found
You used a variable before creating it. Use set x to ... first.
Cannot divide by zero
The number you're dividing by is 0. Check your variable value.
List index out of range
Lists start at index 0. If your list has 3 items, valid indexes are 0, 1, 2.
Key not found in dict
That key doesn't exist yet. Use set key first.
Function not defined
You're calling a function that doesn't exist. Define it first with define ... end define.
Infinite loop risk
Your while loop may never stop. Make sure the condition eventually becomes false.
